DataManager - служба, которая извлекает данные о пользователях, генерирует на
основе этих данных xml файл(и xsd схему к нему) необходимой структуры, после
чего отправляет его на FTP сервер(WindowsService1), разработанный в предыдущих
лабораторных. DataManager,используя ConfigurationManager из 3 лабораторной, 
получет нужные настройки, среди которых путь наначения Source папки для FTP
сервера и строку подключения к нашей базе данных. После этого строка подключения
отправлеятся на ServiceLayer, который в свою  очередь передает ее DataAccessLayer.
Он, используя хранимую процедуру, извлекает информацию из базы данных и помещяет 
ее в список обьектов Person. Этот список возвращается на уровень ServiceLayer, где 
его напосредственно передают XmlGenerator'у, после чего он генерирует xlm и xsd файлы. 
На завершающем этапе DataManager отправляет полученные файлы на FTP сервер, который
проводит с ними манипуляции из прошлых лаб.
